# Mini-Projeto-HealthSync---Plataforma-de-Telemedicina-Inteligente-

Scaffolding de documentação para o mini-projeto **HealthSync - Plataforma de Telemedicina Inteligente**.

## Estrutura do repositório

- `Template_Aluno_HealthSync.md`: template original consolidado.
- `docs/`: documentação organizada por ciclo.
  - `ciclo-1/`: visão e requisitos.
  - `ciclo-2/`: blueprint e decisões.
  - `ciclo-3/`: cloud e resiliência.
  - `bonus/`: evolução arquitetural (opcional).
  - `referencias.md`: referências bibliográficas.

## Como usar

Preencha cada arquivo conforme os ciclos do projeto, anexando diagramas nos pontos indicados e registrando decisões arquiteturais na pasta de ADRs.
